### 1. Daily Backups
- Every form submission creates/updates a daily JSON file
- Format: `contacts_YYYY-MM-DD.json`
- Contains all submissions for that day
- Human-readable JSON format

### 2. Monthly Archives
- Created automatically at month end
- Combines all daily files for the month
- Format: `contacts_YYYY-MM.json`
- Stored in `archive/` folder
